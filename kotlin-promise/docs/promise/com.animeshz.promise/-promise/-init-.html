Promise(resolver: (resolve: (Any?) -> Any?, reject: (Throwable) -> Any?) -> Any?, canceller: (() -> Any?)? = null)

Constructs a Promise and start the resolver, after execution of resolver the state of promise is transitioned to either PromiseState.FULFILLED or PromiseState.REJECTED.

Parameters

resolver - a lambda which controls the state of promise and responsible for the work function of the promise.

canceller - optional parameter, a lambda which is called before cancelling the promise if cancel is called.
