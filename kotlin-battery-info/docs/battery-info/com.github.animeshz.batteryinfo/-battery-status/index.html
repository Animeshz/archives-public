BatteryStatus

data class BatteryStatus

Parameters

isAcConnected - Is external power supply connected?

isBatteryPresent - Is the battery present

batteryFlow - Is the battery BatteryFlow.Charging or BatteryFlow.Discharging or the state is BatteryFlow.UNKNOWN

maxCapacityMWh - Max capacity of the battery in mWh

currentChargeMWh - Current charge in the battery in mWh

flowRateMWps - Flow rate of the battery +ve when charging -ve when discharging, in mW/s

timeToEmptyFromOS - Time to empty calculated from OS, might be zero (0) if not mentioned.

timeToEmpty - Time to empty calculated by flowRateMWps and currentChargeMWh. It is -1 when battery is charging.

timeToFull - Time required to complete charge the battery, calculated via flowRateMWps and currentChargeMWh. It is -1 when discharging.

currentChargePercent - Current charge percent (accurate) calculated from currentChargeMWh and maxCapacityMWh.

currentChargeIntegralPercent - Current charge reported by the OS (Int).
